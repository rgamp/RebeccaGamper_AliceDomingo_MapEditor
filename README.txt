Visual studio 2018

Installer monogame http://community.monogame.net/t/monogame-3-7-1-release/11173

Lancer le programme 

Pour faire apparaître l'éditeur de map, appuyer sur TAB (possibilité de le faire redisparaitre en réappuyant sur tab)

Cliquer sur les sprites en haut de l'éditeur de map, puis recliquer à l'endroit désirer dans la map dans l'éditeur de map juste en dessous.

L'édition s'affiche en direct dans la preview sur le côté droit.
